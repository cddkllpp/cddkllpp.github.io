Java

  • 数据转换: 数据转换就是将一种数据类型转换为另一种数据类型，比如int–》long。分为自动转换和强制转换。
    自动转换：当数据类型转换是由小范围数据转大范围数据时为自动转换。例如 long num = 100；就是将默认的int转化为long。
    强制转换：当转换为大到小时就不能自动转换了，必须的进行强制转换，其格式为范围小的类型 范围晓得变量名 = （范围小的类型） 原本范围大的数据；但是不推荐使用强制转换，因为会出现数据溢出或者精度损失。
  • 编码表;计算机底层为01数字，所以所有的字符都是由数字组成，而数字与字符对照的表叫做编码表，分为ASCII和UNICOD.
